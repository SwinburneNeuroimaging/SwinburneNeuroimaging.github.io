    Accessing the Supercomputer

    OzSTAR and NT are Swinburne's two supercomputers, they are designed to help Australian researchers analyse large amounts of data and run complex calculations that would be impossible on a regular computer or laptop.

    To access neuroimaging data (MRI/DICOM, MEG/FIF) obtained at SNI or if your study requires supercomputing resources for data analysis, such as running analysis using Freesurfer, MRTrix, fMRIPrep, mne-python pipelines, then you can access Swinburne's supercomuters using the following instructions.

    Tip

    If you don't yet have an OzSTAR/NT account, you can apply for one and create or join a project on the OzSTAR account portal and by following the Creating an OzSTAR/NT Account tutorial. Your username can be found on the portal, and your password can also be reset there.

    Access OzSTAR/NT via the terminal (MacOS/Linux/Windows)

    MacOS and Linux both have their build-in applications (Terminal for MacOS, Console for Linux) for command line.

    Open Finder → Applications → Utilities → Terminal, or press Cmd + Space and type Terminal.

    Press Ctrl + Alt + T, or search for Console or Terminal in your applications menu.

    Windows has two options:

    PowerShell is built in to Windows. Search for it in the Start menu. You can also download and install a newer version if needed.

    WindTerm is a free cross-platform SSH client with a more user-friendly interface. You can download it here.

    Login

    Once you have a terminal open, connect to OzSTAR using SSH with your username: 

    $ ssh username@ozstar.swin.edu.au
    To connect to Ngarrgu Tindebeek (NT) instead: 
    $ ssh username@nt.swin.edu.au

    Note

    ozstar.swin.edu.au and nt.swin.edu.au both provide access to the same data, but with have different computing resources.

    If you want to learn more about SSH, you can read our SSH basics tutorial.

    Login nodes

    When you connect, you will land on one of the login nodes:

    System Login nodes
    OzSTAR farnarkle1, farnarkle2
    NT tooarrana1, tooarrana2

    This is where your home directory will be on OzSTAR or NT. This is usually /home/Username/. You can do light calculations and commands here.

    Warning

    Login nodes are shared by all users and are intended for light tasks only, file management, writing scripts, and submitting jobs. Do not run computationally intensive work directly on a login node. Use the job submission system instead. To learn how to submit a job, read our Submitting Jobs to OzSTAR/NT tutorial.

    Data location: /fred/ozXXX/

    Your project data will be located at /fred/ozXXX/ where ozXXX is your OzSTAR/NT project number. 

    $ cd /fred/ozXXX/
    We recommend creating a symlink (a shortcut) from your home directory to your project data directory. 
    $ cd ~                        # Make sure you are in your home directory
$ ln -s /fred/ozXXX/ ozXXX    # ozXXX is your project number
    You should now find a new file in your home directory called ozXXX. You can $ cd ozXXX to get directly to your project data.

    Transferring files with FileZilla

    If you prefer a graphical interface for accessing and downloading data, FileZilla is a free file transfer program that can connect to OzSTAR/NT over SSH. You can drag and drop files between your local machine and OzSTAR without using the command line.

    See the Nagivating the supercomputer and downloading data guide for detailed instructions on how to install and connect FileZilla to OzSTAR.

    Remember to change the Remote site to /fred/ozXXX to see your data.

    Submitting jobs

    To run an analysis on OzSTAR, you submit a job , a script that describes what you want to run and what computing resources you need. See the Submitting Jobs to OzSTAR/NT guide for a full walkthrough.
