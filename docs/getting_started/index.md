# Getting Started with SNI Resources

Welcome to the Swinburne Neuroimaging resources guide. This documentation will walk you through the process of contacting us, setting up accounts and accessing high-performance computing resources for neuroimaging research.

``` mermaid
flowchart LR
    A([Contact us]) --> B([Submit application])
    B --> C([Create OzSTAR account])
    C --> D([Join your project])
    D --> E([Ready to go])

```

## Step 1: Get in touch!
We can support you at all stages of your research, including:

- Project conception and design
- Logistical planning
- Advice on research ethics
- Advice on data management
- Expert training in data analysis

Before doing anything else, reach out to discuss your project with the team.
We'll help you figure out which facilities and resources are right for you.

Email us to start the conversation: [neuroimaging@swinburne.edu.au](mailto:neuroimaging@swinburne.edu.au)

!!! tip "Not sure what you need?"
    That's fine, send us a brief description of your research question and we'll suggest the most appropriate modality and workflow.


## Step 2: Submit your application
Once you've spoken with the team, you'll be directed to our online application form. The form covers:

- [EEG & TMS project application form](https://swinburneuniversity-2.snapforms.com.au/form/sni-project-application-eeg-and-tms)

- [MRI & MEG project application form](https://swinburneuniversity-2.snapforms.com.au/form/sni-project-application-meg-and-mri)

!!! note "For these forms you will need:"
    - Original ethics application*
    - Evidence of ethics approvals*
    - Evidence of amendment/s to the ethics application (and approvals)
    - Participant Information and Consent Form/s indicating how incidental findings will be managed*
    - Study protocol information*
    - Funding information

## Step 3: Setting up OzSTAR Supercomputing
After you have setup a project with us, you can follow these tutorials to set up you OzSTAR/NT supercomputer access.

1. [Create an OzSTAR account](../computing_resources/tutorials/supercomputing/create_ozstar_account.md)
2. If you are already part of a project (i.e a student or collaborator of someone elses project) you can [Join an OzSTAR/NT project](../computing_resources/tutorials/supercomputing/join_an_ozstar_project.md)
3. [Set Up Neurodesk](../computing_resources/tutorials/neurodesk/setting_up_neurodesk.md)
4. [Connect Neurodesk to OzSTAR/NT](../computing_resources/tutorials/neurodesk/connect_neurodesk_to_ozstar.md)
