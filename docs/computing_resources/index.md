# Computing Resources

## [Don't know where to start? Follow this walkthrough](./tutorials/computing_walkthrough.md). 
This is a curated collection and ordering of tutorials and documentation to step you through all the information in an order that makes sense, and builds from the previous step. You can choose your own starting point, or dive into a page you are most interested in.

## [Tutorials](./tutorials/index.md)
Tutorials are step-by-step guides designed to be followed along at your own pace. They are aimed at getting you up and running with the tools and systems used at SNI. Some currently available tutorials include an introduction to the Linux command line, SSH and connecting to remote servers, and getting started with Neurodesk.

## [Documentation](./documentation/index.md)
Documentation is reference material you can come back to when you need it. It is not intended to be followed as instructions, but can be more detail. Some currently documentation includes an overview of Neurodesk, an overview of OzSTAR and NT (Swinburne's supercomputers), and a BIDS reference guide.
