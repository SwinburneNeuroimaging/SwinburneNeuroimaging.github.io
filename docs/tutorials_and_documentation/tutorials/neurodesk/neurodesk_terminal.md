# Opening a Terminal in Neurodesk

## What is a Terminal?

A terminal (also called a command line or shell) is a text-based interface where you can type commands to interact with the computer. Instead of clicking buttons and icons, you type instructions that the computer executes.

Don't worry if you've never used a terminal before, this guide will show you exactly what to do!

---

## How to Open a Terminal

**Step 1: Locate the terminal icon**

Look at the bottom of your Neurodesk desktop screen. You'll see a taskbar with several icons.

![Neurodesk Desktop](/assets/images/tutorials/terminals_neurodesk/neurodesktop_with_arrow_to_terminal_icon.png)

**Step 2: Click the terminal icon**

Click on the terminal icon in the taskbar (it looks like a small black rectangle or screen).

**Step 3: The terminal window opens**

A new window will appear with a black background and white text. This is your terminal!

![Terminal Window](/assets/images/tutorials/terminals_neurodesk/neurodesktop_with_terminal.png)

You'll see a command prompt (text ending with `$` with a white bar next to it) waiting for you to type commands.

---

## Using the Terminal

Once your terminal is open:

1. **Type or paste commands** exactly as shown in the instructions
2. **Press ENTER** to execute each command
3. **Wait for each command to complete** before entering the next one

!!! tip "Copying and Pasting"
    You can copy commands from the documentation and paste them directly into the terminal:
    
    - **Copy**: Select text and press `Ctrl+C` (or right-click and select Copy)
    - **Paste into terminal**: Right-click in the terminal window and select Paste, or press `Ctrl+Shift+V`

!!! note "What's That Text?"
    The text you see when the terminal opens (something like `vdiuser@vdn-xbstby: ~$`) is your command prompt. It shows:
    
    - Your username (`vdiuser` in this example)
    - The computer name (`vdn-xbstby` in this example)
    - Your current directory (`~` this is the 'home' directory of the computer)
    - The `$` symbol means the terminal is ready for your command

---

## You're Ready!

That's it! You now know how to open a terminal in Neurodesk. You can proceed with the connection instructions in the [Neurodesk setup guide](/Getting Started/setting_up_neurodesktop.md).

If you need to open additional terminal windows at any time, just click the terminal icon again.
